# 项目简介

## 核心功能
- 基于 YOLOv8 实现的黄瓜植株害虫检测系统
- 支持图片、视频及摄像头进行实时检测
- 提供 4286 张黄瓜植株害虫数据集，包含 5 类害虫（蚜虫、果蝇、南瓜甲虫、潜叶蝇、粉虱）
- 支持单张图片、图片文件夹、视频文件和本地摄像头的检测
- 实时显示目标位置、目标总数、置信度等信息
- 支持自定义置信度和 IOU 阈值
- 支持选择显示标签和原图
- 支持选择检测模型
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存
- 提供完整程序文件、UI界面源文件、图标、数据集图片和项目运行教程

## 适用场景
- 农业种植中的害虫监测与防治
- 植物保护研究
- 农业自动化生产管理
- 害虫识别与计数分析

## 技术特点
- 采用 PyCharm+Anaconda 或 VSCode+Anaconda 开发环境
- 基于 Python 3.9，依赖 opencv-python、PyQt5、ultralytics、torch1.9 等库
- 已训练好模型，配置成功环境可直接使用
- 提供完整源码、标注数据集、训练好的模型及训练结果
